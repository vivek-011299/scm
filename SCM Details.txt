Smart Contact Manager
1. User signup with email and password.
2. Verify account using email verification link/with otp
3. user signup with google and GitHub
4. Add contact with picture
5. Picture stored in aws/cloudinary
6. user can view all contact
7. user can view one contact detail
8. Send an email directly to a contact from our manager app
9. email contains text or attachments etc.
10. update the contact
11. Delete the contact
12. search the contact
13. Pagination
14. Export contacts to excel
15. Mark favorite contacts
16. see and edit own profile details
17. Provide feedback.
18. Dark light theme
